Başlarken

HTML, JavaScript, CSS gibi diller "betik diller" gurubunda yer alırlar ve doğrudan işletim sistemi üzerinde çalışmazlar. 3.party yazılımlara ihtiyaç duyarlar (internet tarayıcısı, JavaVM – Java Virtual Mavhine) gibi. Bu, kodlarımızı ancak bu programlar üzerinde çalıştırabileceğimiz anlamına gelir. Bu nedenle JavaScriptte yazdığımız kodaları internet tarayıcımız üzerinde çalıştırmak zorundayız ve bunun için de bir web sayfasıına ihtiyacımız olacak. Bir web sayfasının nasıl oluşturalacağını öğrenmek için HTML bölümümüzü takip edebilirsiniz.

O halde bir web sayfasına Javascript kodlarımızı nasıl ekleyeceğimizi öğrenerek başlayalım.

<script> etiketi

JavaScript kodlarımızı bir HTML sayfasına eklemek için kullanabileceğimiz iki yöntem var.

Doğrudan <script> …… </script> tagları arasına yazmak,

Harici bir script dosyası içerisine yazmak ve bu dosyayı HTML sayfamız içerisine eklemleyerek çalışmasını sağlamak.

Javascript programları html içerisine <script> etiketi ile eklenebilir. Örneğin:

<!DOCTYPE HTML>
<html>
  <body>
   <p>Javascript Kodundan önce...</p>
   <script>
    alert( 'Hello, world!' );
   </script>
   <p>...Javascript kodundan sonra.</p>
  </body>
</html>

<script> etiketi içerisine yazdığınız JavaScript komutu tarayıcı o koda geldiğinde doğrudan okunur.

Dışardan yüklenen Javascript kod dosyaları

Eğer çok fazla JavaScript kodunuz varsa bunları ayrı bir dosya içerisine yazıp uzantısı .js olacak şekide kaydedebilir ve sonrasında bu dosyayı aşağıdaki gibi yol göstererek sayfanızda çalıştırılmasını sağlayabilirsiniz.

<!DOCTYPE HTML>
<html>
  <body>
   <script src="script.js"> </script>
  </body>
</html>

Dikkate değer:

Kural olarak en basit JavaScript kodları doğrudan HTML içerisine yazılır. Daha karmaşık olanlar farklı dosyalarda taşınır. Dikkat edilmesi gereken şey, src parametresi kullanarak oluşturduğunuz <script src=”script.js></script> tagları arasına yerleştireceğiniz kodlarınızın çalışmayacak olmasıdır. Örneğin aşağıdaki kod bloğu çalışmayacaktır:

<!DOCTYPE HTML>
<html>
  <body>
   <script src="script.js">
    alert(1); // İçerik görmezden gelinecektir çünkü "src" tagı kullanılmıştır.
   </script>
  </body>
</html>

Böyle bir durumda sayfa içerisinde başka kodlar kullanma ihtiyacı oluşursa yeni bir <script> tagı açılmalıdır. Yani doğru yazım şekli aşağıdaki gibi olmalıdır.

<!DOCTYPE HTML>
<html>
  <body>
   <script src="script.js"> </script>
   <script>
    alert(1);
   </script>
  </body>
</html>
